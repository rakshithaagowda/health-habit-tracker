# 🏃‍♀️ Health Habit Tracker

A modern and interactive **React-based web application** designed to help users build, track, and maintain healthy daily habits with ease.

---

## 🌟 Overview

The Health Habit Tracker allows users to add daily habits, mark them as completed, and monitor their consistency over time. It promotes self-discipline and healthy lifestyle building through a simple and intuitive interface.

---

## 🚀 Features

- ➕ Add new daily habits  
- ✅ Mark habits as completed  
- 📊 Track daily progress  
- 🧹 Clean and minimal UI  
- ⚡ Fast performance using Vite + React  
- 📱 Responsive design (works on all devices)

---

## 🛠️ Tech Stack

- React.js  
- JavaScript (ES6+)  
- HTML5  
- CSS3  
- Vite

## ▶️ How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/rakshithaagowda/health-habit-tracker.git
```

### 2. Navigate to project folder
```bash id="nav1"
cd health-habit-tracker
```

### 3. Install dependencies
```bash id="inst1"
npm install
```

### 4. Start development server
```bash id="start1"
npm run dev
```

### 5. Open in browser
Open the link shown in the terminal.
